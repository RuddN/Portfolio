Dice Lab

Project Struggles:

For me, the hardest part of the dice lab was figuring out how to align all the dots correctly within the dice without having to code each dot individually for each dice based on its value. I discovered that I could utilize previously coded values and overlap them to save myself a lot of time. See the code below. :)

Favorite Section of Code:

void one(){
fill(0,0,0);
ellipse(midx, midy, 7, 7);
}

void two(){
fill(0,0,0);
ellipse(topx, topy, 7, 7);
ellipse(botx, boty, 7, 7);
}

void three(){
fill(0,0,0);
one();
two();
}

void four(){
fill(0,0,0);
two();
ellipse(botx, topy, 7, 7);
ellipse(topx, boty, 7, 7);
}

void five(){
fill(0,0,0);
one();
four();
}

void six(){
fill(0,0,0);
four();
ellipse(topx, midy, 7, 7);
ellipse(botx, midy, 7, 7);
}
